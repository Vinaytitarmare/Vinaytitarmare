## Featured Projects

### 🔬 SnapMind — AI Research Assistant
> *Edge Extension · React · React Native · Python · RAG · Cohere*

An AI-powered research tool for professors and students that builds interactive **Entity-Relationship graphs** to map correlations across complex topics. Features an intelligent browser agent (via Firecrawl) that scrapes the web to synthesize answers with auto-generated citations, a visual Q&A mode for image-heavy documents, and state persistence for continuous research sessions.

---

### 🔖 SnapBook — Unified Cross-Platform Bookmarking
> *React · React Native · Python · LLMs · Browser Extension*

A bookmarking app that unifies saved links across browser extension and mobile share intents into one central hub. Eliminates context loss from media fragmentation and reduces re-watching friction by augmenting links with **AI-generated summaries and searchable tags**.
